call 和 apply 方法

这两个方法可以动态改变对应对象里面的 this 指向

也就是说当某个对象 (A) 没有某个方法,但是其他的对象 (B) 有,没有就可以借助call 和 apply 方法 (改变对象B里面this的指向,让B里面的 this指向A) 来实现,让对象(A)直接使用对象(B)里面的方法
