JS数组常用方法

发表于 2019-08-14 更新于 2021-07-13 分类于 前端 ， JavaScript

通过索引访问元素

forEach 遍历数组

some 遍历数组

用途

遍历数组中每个元素，判断其是否满足指定函数的指定条件,返回true或者false

  • 如果一个元素满足条件,返回true,且后面的元素不再被检测
  • 所有元素都不满足条件，则返回false
  • 不会改变原始数组
  • 不会对空数组进行检测;数组为空的话，直接返回false
